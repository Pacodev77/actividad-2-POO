# Actividad de Programación Orientada a Objetos: Clase Carro

Este repositorio contiene un ejemplo de implementación de la clase `Carro` en Python, demostrando conceptos básicos de programación orientada a objetos como encapsulamiento, abstracción y métodos.

## Descripción

La clase `Carro` permite crear objetos que representan automóviles con atributos como marca, modelo, color y costo. Además, incluye métodos para acceder al costo (`get_costo`) y para mostrar la información del carro (`__str__`).

## Estructura del código

- `carro.py`: Contiene la definición de la clase `Carro` y ejemplos de su uso.
